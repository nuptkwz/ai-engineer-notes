## AI应用开发工程师笔记

欢迎来到「AI应用开发工程师笔记」仓库！这里系统地整理和汇集了我在学习、实践AI应用开发，特别是大模型相关技术过程中的**实战笔记、项目心得和解决方案**。本仓库旨在为同样致力于AI应用开发的同行和爱好者提供一个**
高质量、重实践、持续更新**的参考知识库。

## ✨ 仓库特色

* **🎯 实战导向**：笔记内容源于真实项目场景和实验，聚焦于解决AI应用开发中的实际问题，而非纸上谈兵。
* **🚀 技术前沿**：紧密跟踪大语言模型（LLM）、AI Agent、RAG（检索增强生成）等前沿技术，并提供经过验证的实践方案。
* **🔧 即学即用**：包含大量可运行的代码片段、详细的配置说明和清晰的架构图，助力快速上手和复现。
* **📚 体系化梳理**：内容结构化编排，涵盖从基础概念、核心原理到高级应用和性能优化的完整学习路径。
* **💡 持续进化**：随着AI技术的快速迭代，本仓库将定期更新，努力保持内容的前瞻性和实用性。

## 📖 内容导览

本仓库的笔记主要涵盖以下核心领域：

### 1. 大语言模型基础与应用

* LLM 核心原理与微调实战（LoRA, P-Tuning等）
* 主流大模型（GPT系列、Claude、国产模型）API应用详解
* 提示词工程（Prompt Engineering）最佳实践与案例库

### 2. AI应用开发框架与工程化

* LangChain/LlamaIndex 核心概念与高级应用案例
* AI Agent 开发实战：从ReAct模式到多智能体系统
* 模型服务化部署：FastAPI、Docker、模型量化与加速

### 3. RAG系统构建与优化

* RAG 全链路详解：从文档解析、向量化到检索与生成
* 向量数据库（Chroma, Pinecone, Milvus）选型与实战
* RAG 性能优化与评估指标

### 4. 行业解决方案与项目实战

* 基于大模型的行业应用案例（如智能客服、内容生成、代码助手等）
* 端到端项目复盘：技术选型、架构设计与踩坑记录

### 5. 工具链与资源

* 开发环境配置、常用工具脚本、调试技巧合集
